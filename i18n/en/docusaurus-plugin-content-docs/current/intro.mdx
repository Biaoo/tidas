# Introduction

TIDAS (**TI**angong **DA**ta **S**ystem) is an open-source JSON-based data system that provides standardized solutions for lifecycle data management.

## Core Foundations

The fundamental bases of TIDAS:

- **[LCA/Carbon Footprint Database Guidelines](https://www.carbonfootprint.network/docs/category/lca-database-guideline)** - Fully compliant with [ISO 14040/44](https://www.iso.org/home.html), [ILCD Handbook](https://eplca.jrc.ec.europa.eu/ilcd.html) and standards like [ecoinvent](https://www.ecoinvent.org/).
- **[JSON-based Data Structure](/docs/category/tidas-schema)** - Built on JSON structure with JSON Schema validation, offering high extensibility and compatibility to support future technical architectures (blockchain, data spaces, AI).
- **[Multilingual Core Data Package](https://github.com/linancn/EF-reference-package-3.1-Correction)** - Provides standardized foundational databases ensuring data consistency and traceability.

## Core Tools

Based on these foundations, we've developed the following tools (see [TIDAS Toolkit](/docs/category/tidas-tool)):

- **`validate`** - **Data Validation Tool**: Ensures data quality and compliance with specifications
- **`convert`** - **TIDAS-eILCD Bidirectional Converter**: Enables lossless conversion with formats like eILCD
- **`export`** - **Data Export Tool**: Supports multiple format exports and data sharing
