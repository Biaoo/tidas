JSON Schema for Flow and Flow categories
========================

:::tip
Flow datasets contain invariant information about substances or energy forms, aligned with the "inputs and outputs" section of ISO/TS 14048. Each flow represents a physical exchange (e.g., material, energy, or waste), including identifiers and classifications. Flows are the building blocks of process and system datasets, ensuring consistency and traceability.

:::
