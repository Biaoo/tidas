# 简介

TIDAS（TianGong LCA Data System）是一个基于JSON的开源数据结构体系，提供了标准化、可扩展的数据管理解决方案。

TIDAS简化了生命周期数据管理流程：

  **方法层**: 生命周期数据管理遵循**[LCA/碳足迹数据库构建导则](https://www.carbonfootprint.network/docs/category/lca-database-guideline)**,完全兼容[ISO 14040/44](https://www.iso.org/standard/37456.html)，[ILCD Handbook](https://eplca.jrc.ec.europa.eu/ilcd.html)以及[ecoinvent](https://www.ecoinvent.org/)等标准。

  **数据层**: TIDAS采用统一的数据格式，基于JSON结构和JSON Schema验证，具有高度可扩展性和兼容性，可以支持未来全域技术架构（区块链、数据空间、AI）。

  **转换层**: TIDAS和eILCD具有多语言双向无损映射服务，实现数据格式无损转换，支持与eILCD等主流格式互通，让不同系统间的数据交换变得简单高效。

  **平台层**: **[TianGong LCA平台](https://docs.tiangong.earth/)**可以实现LCA数据的研制和发布需求，支持开放平台和私有部署两种模式，满足不同机构的应用需求。
