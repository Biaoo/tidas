模型数据集的数据结构
========================


:::tip
模型数据集描述了一个产品系统的完整或部分生命周期，由多个相互连接的过程组成。基于指定的“基准过程”，将各个过程通过产品和废弃物流连接起来，形成一个系统化的生命周期模型。​这有助于在生命周期评估中准确追踪和量化产品系统的环境影响。

:::
