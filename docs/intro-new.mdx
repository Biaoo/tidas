# TIDAS 简介

TIDAS（**TI**angong **DA**ta **S**ystem）是一个基于JSON的开源数据系统，为生命周期数据管理提供标准化解决方案。

## 底层基础

TIDAS的核心基础：

1. **科学的方法** - 遵循遵循[LCA/碳足迹数据库构建导则](https://www.carbonfootprint.network/docs/category/lca-database-guideline),完全兼容[ISO 14040/44](https://www.iso.org/home.html)，[ILCD Handbook](https://eplca.jrc.ec.europa.eu/ilcd.html)以及[ecoinvent](https://www.ecoinvent.org/)等标准。
2. **面向未来的数据结构** - 基于JSON结构和JSON Schema验证，具有高度可扩展性和兼容性，可以支持未来全域技术架构（区块链、数据空间、AI）。
3. **一致的基础流编码（UUID）清单** - 提供标准化的核心底层基础数据库，确保数据的一致性和可追溯性

## 核心工具

基于上述基础，我们开发了以下工具：

1. **TIDAS-eILCD双向转换器** - 实现与eILCD等主流格式的无损转换
2. **数据导出工具** - 支持多种格式的数据导出和共享
3. **数据验证工具** - 确保数据质量和符合规范要求

## 集成解决方案

我们构建了以下集成应用方案：

1. **天工LCA平台** - 全功能数据研制和发布平台，支持私有部署
2. **区块链解决方案** - 实现数据不可篡改和可追溯
3. **AI集成解决方案** - 支持智能数据分析和处理
