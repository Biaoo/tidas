# MCP AI 服务

MCP AI 服务是 TIDAS 系统中基于 AI 的智能引擎组件，旨在提升生命周期数据的理解、补全与校验能力，适配新一代环境数据管理与智能应用场景。

## ✨ 核心功能

MCP（Multi-channel Cognitive Processor）集成了 AI 能力至 TIDAS 生命周期数据库中，提供如下核心服务：

- **语义搜索**：通过自然语言查询生命周期清单（LCA Flows）数据
- **智能补全**：辅助补充缺失字段或建议结构完善
- **格式校验**：基于 JSON Schema 自动校验数据合规性
- **人机交互界面**：通过 MCP Inspector 可视化操作和调试流程（见下图）

## 🧱 技术实现

MCP 服务通过以下方式实现：

- 传输协议：支持 SSE、WebSocket、HTTP 等模式
- 数据处理：基于 TIDAS 数据格式（JSON Schema）进行解析
- 工具调度：集成 `Search_flows_Tool` 等多种智能工具
- 本地化部署：支持本地私有部署，保障数据安全

## 🔗 开源链接

MCP 服务开源于 GitHub：  
👉 [https://github.com/linancn/tiangong-lca-mcp](https://github.com/linancn/tiangong-lca-mcp)

该项目提供了完整的服务端源码、Inspector 前端、工具注册与测试文档，可用于快速构建 AI 驱动的 LCA 智能平台。

## 📦 使用前提

- 已构建 TIDAS 数据库，结构符合 [TIDAS Schema](/docs/category/tidas-schema)
- 配置 MCP Bearer Token 与传输地址
- 按需注册所需工具（如 Search_flows_Tool）
